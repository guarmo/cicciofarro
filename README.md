It was developed with the following tools:
<ul>
  <li>NextJS</li>
  <li>Chakra-ui</li>
  <li>EmailJS</li>
  <li>React-icons</li>
  <li>React-player</li>

## Installing cicciofarro.com

To install cicciofarro.com, follow these steps:

Linux and macOS:
```
npm install
```

Windows:
```
npm install
```
